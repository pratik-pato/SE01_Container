This directory contains
2 sub directory and 3 seperate python scripts which are responsible for following task

a]    genTableList.py : takes database name as argument and generates genTableList.html which gives list of relations in database

b]    genColList.py : takes database name and table name as argument and generates genColList.html which gives list of columns in relations

c]    genTable.py : takes database name and table name as argument and generates genTable.html which gives editable table.
